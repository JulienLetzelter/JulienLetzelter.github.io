# Tempo Website (GitHub Pages)

Static bilingual website (English/French) for the iOS app Tempo.

## Files

- `index.html`: marketing homepage
- `privacy.html`: full privacy policy page (EN/FR)
- `styles.css`: visual style
- `script.js`: language switch + screenshots gallery
- `assets/tempo-logo.png`: logo
- `assets/screen_EN/*` and `assets/screen_FR/*`: app screenshots

## Publish on GitHub Pages

1. Create a GitHub repository named `JulienLetzelter.github.io` (or push this folder to that existing repository).
2. Push the content of this folder to the repository root.
3. In GitHub, open **Settings > Pages**.
4. In **Build and deployment**, choose:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or `master`)
   - **Folder**: `/ (root)`
5. Save and wait 1-2 minutes.
6. Your site should be live at `https://julienletzelter.github.io/`.

## Notes

- The website auto-detects browser language and supports manual toggle `EN/FR`.
- The privacy policy text matches the provided legal source content.
